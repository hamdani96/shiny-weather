Shiny-weather app

Aplikasi Shiny ini dirancang untuk memvisualisasikan data cuaca dari weather.csv secara interaktif. Pengguna dapat memilih variabel yang ingin dianalisis serta menentukan jenis visualisasi yang diinginkan, seperti scatter plot, line plot, bar plot, atau melihat tabel datanya secara langsung. Setiap pilihan variabel akan menyesuaikan input secara otomatis, sehingga memudahkan proses eksplorasi data. Grafik-grafik yang ditampilkan bersifat interaktif karena dibuat menggunakan plotly, sehingga pengguna dapat melakukan zoom, hover, serta membaca detail nilai dengan mudah. Secara keseluruhan, aplikasi ini membantu pengguna memahami hubungan antar variabel dalam dataset cuaca dengan cara yang intuitif, fleksibel, dan informatif.
